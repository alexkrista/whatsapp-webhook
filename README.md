# KRISTA Baustellenmanager

## Version

Aktueller Entwicklungszweig: **build-0014**

## Projekt

KRISTA ist ein digitaler Baustellenmanager für Handwerksbetriebe.

Bestandteile:

- Baustellenverwaltung
- Mitarbeiterverwaltung
- WhatsApp-Assistent "Kristine"
- Zeiterfassung
- Regieberichte
- Baustellenprotokolle
- PDF-Erstellung
- Chef-Dashboard
- Stundenkontrolle

---

## Projektstruktur

```
public/
server.js
kristine.js
morning-status.js
pdf.js
package.json
```

---

## Entwicklungsregeln

- `main` enthält immer eine lauffähige Version.
- Jede größere Änderung erfolgt in einem eigenen Build-Branch.
- Vor jedem Merge werden alle Funktionen getestet.
- Keine halbfertigen Funktionen auf `main`.
